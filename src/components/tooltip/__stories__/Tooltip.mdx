    # Tooltip

    A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

    ## **Purpose and Context**

    Tooltips are transient views that appear when a user interacts with an element. They provide context, help, or extra information without cluttering the UI.
    They are commonly used to:

    -   Label icons or buttons that don't have text labels.
    -   Provide additional context or explanations for specific terms.
    -   Display full text for truncated content.
    -   Offer shortcuts or keyboard commands for an action.

    In {config.title}, tooltips are customizable and accessible, built on top of Radix UI:

    -   **Content**: The message or element to display inside the tooltip.
    -   **Side**: Controls the positioning of the tooltip relative to the trigger (top, right, bottom, left).
    -   **Theme**: Switches between light and dark visual styles.
    -   **Delay Duration**: Adjusts how long the user must hover before the tooltip appears.
    -   **Size**: Controls the padding and text size of the tooltip.

    ## **FAQ**

    #### **How do I change the position of the tooltip?**

    Use the `side` prop to specify the preferred side (`top`, `right`, `bottom`, `left`).
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Tooltip content="I'm on the right" side="right">
          <button>Hover me</button>
        </Tooltip>
      `} />
    </Canvas.Source>

    <br />
    #### **How do I change the color theme?**

    Use the `theme` prop to switch between `dark` (default) and `light` modes.
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Tooltip content="Light Mode" theme="light">
          <button>Hover me</button>
        </Tooltip>
      `} />
    </Canvas.Source>

    <br />
    #### **How do I adjust the hover delay?**

    Use the `delayDuration` prop to set the time in milliseconds before the tooltip appears.
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Tooltip content="Fast tooltip" delayDuration={100}>
          <button>Hover me</button>
        </Tooltip>
      `} />
    </Canvas.Source>
