    # Skeleton

    A component used to display a placeholder preview of content while it is loading.
    <DocInfo 
      importStatement={`import { Skeleton } from 'clera-ui/skeleton'`} 
      sourcePath={`components/skeleton`}

    <br />

    ## **Purpose and Context**

    Skeletons are used to create a "content-first" loading experience, reducing layout shift and providing a better perception of performance compared to a traditional spinner. They should mimic the shape and size of the content that will eventually load.

    -   **`className`**: Use utility classes (e.g., `h-4`, `w-full`) to control the size and shape.
    -   **`rounded`**: Use variants to match the border-radius of the content you are loading (e.g., `rounded="full"` for an avatar).
