    ## **Purpose and Context**

    Cards are versatile UI elements used to organize and present information in a concise and visually appealing manner. They are commonly used to:

    -   Display summaries of articles or blog posts.
    -   Showcase product listings or services.
    -   Present user profiles or team member information.
    -   Group related actions or options.

    In {config.title}, cards are highly customizable and can be adapted to different contexts through their props:

    -   **Title**: Provides a clear heading for the card's content.
    -   **Description**: Offers a brief summary or overview of the card's content.
    -   **Thumbnail**: Displays an image to visually represent the card's content.
    -   **Rounded Size**: Controls the roundness of the card's corners.
    -   **Padding**: Adjusts the spacing within the card.
    -   **ClassName**: Allows for custom styling with Tailwind CSS classes.
    -   **Link**: Makes the entire card clickable, navigating to a specified URL.
    -   **Meta Items**: Displays additional information or metadata.
    -   **Footer Action Label**: Adds a call-to-action button in the card's footer.
    -   **Text Size**: Controls the text size for the title, description, and footer action.
    -   **Badge**: Adds a badge to highlight specific aspects of the card.

    ## **FAQ**

    #### **How do I add a title and description to a card?**

    Use the `title` and `description` props to specify the title and description text for the card.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Card title="My Card" description="This is a description." />
      `} />
    </Canvas.Source>

    <br />
    #### **How do I add a thumbnail image to a card?**

    Use the `thumbnail` prop to specify the URL of the image to display as a thumbnail.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Card thumbnail="/path/to/image.jpg" />
      `} />
    </Canvas.Source>

    <br />
    #### **How do I add meta items to a card?**

    Use the `metaItems` prop to pass an array of strings to display as meta information.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Card metaItems={["Category", "Date"]} />
      `} />
    </Canvas.Source>

    <br />
    #### **How do I add a footer action to a card?**

    Use the `footerActionLabel` prop to specify the text for the footer action button.  You'll also need to use the `link` prop to specify the URL to navigate to when the button is clicked.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Card footerActionLabel="Learn More" link="/learn-more" />
      `} />
    </Canvas.Source>
