    <br />

    ## **Purpose and Context**

    Badges are used to draw attention to specific elements or pieces of information within an interface. They provide a concise way to convey status, category, or other relevant details. Use them to:

    -   Highlight new or updated content.
    -   Indicate the status of an item (e.g., "In Progress," "Completed").
    -   Categorize items (e.g., "Featured," "Trending").
    -   Display counts or metrics.

    In {config.title}, badges are flexible and can be adapted to different contexts through their props:

    -   **Variant**: Choose a visual style that reflects the meaning of the badge.
    -   **Size**: Adjust the badge size to fit the surrounding content.
    -   **Rounded**: Control the badge's shape with different levels of rounding.
    -   **Icon**: Add an icon to further enhance the badge's visual communication.
    -   **Icon Position**: Place the icon on the left or right side of the badge.

    ## **FAQ**

    #### **What are the different badge variants available?**

    The Badge component supports the following variants: `primary`, `secondary`, `success`, `info`, `warning`, and `destructive`. Each variant provides a different visual style to represent the badge's meaning.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Badge variant="warning">Warning</Badge>
      `} />
    </Canvas.Source>

    #### **How do I change the size of a badge?**

    You can use the `size` prop to control the badge's size. Available sizes are: `sm`, `md`, and `lg`.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Badge size="lg">Large</Badge>
      `} />
    </Canvas.Source>

    #### **How do I control the badge's border radius?**

    Use the `rounded` prop to control the badge's border radius. Options include `none`, `sm`, `md`, and `full`.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Badge rounded="full">Full</Badge>
      `} />
    </Canvas.Source>

    #### **How do I add an icon to a badge?**

    You can add an icon to a badge by passing a React element to the `icon` prop. Use the `iconPosition` prop to control whether the icon appears on the left or right side of the badge label.

    <Canvas.Source>
      <Source className="my-0!" dark code={`
        import { FiUser } from 'react-icons/fi';
        <Badge icon={<FiUser />} iconPosition="left">User</Badge>
      `} />
    </Canvas.Source>
