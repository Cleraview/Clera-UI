    # RadioGroup

    A component for selecting a single option from a set of choices. It is typically used in forms where only one selection is allowed.

    <DocInfo 
      importStatement={`import { RadioGroup } from 'clera-ui/form'`} 
      sourcePath={`components/form/radio`} 
    />

    <br />

    ## **Purpose and Context**

    The `Radio` component is designed to provide users with a clear and intuitive way to select one option from a group. It is commonly used in scenarios such as surveys, preference settings, and any form where exclusive selection is required.

    -  **Checked**: Indicates whether the radio button is selected.
    -  **Disabled**: Disables the radio button, preventing user interaction.
    -  **Label**: The text label associated with the radio button, providing context for the choice.

    In addition to these, the `Radio` component inherits all standard HTML input attributes, allowing for extensive customization and flexibility. This includes attributes such as:
    -  `name`: Groups radio buttons together, allowing only one to be selected at a time.
    -  `value`: The value associated with the radio button when selected.
    -  `onChange`: Handles changes to the selection state.
    -  `required`: Marks the radio button as required for form submission.

    ## **FAQ**
    ### **What are the different states of radio buttons?**
    Radio buttons can be in various states such as checked, unchecked, disabled, and focused.
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <RadioGroup value={value} onChange={setValue} name="billingCycle">
          <Radio 
            label="Checked"
            name="state"
            value="checked"
            checked
          />
          <Radio 
            label="Unchecked"
            name="state"
            value="unchecked"
          />
          <Radio 
            label="Disabled"
            name="state"
            value="disabled"
            disabled
          />
        </RadioGroup>
      `} />
    </Canvas.Source>

    ### **How do I group multiple radio buttons?**
    Use the same `name` attribute for all radio buttons in a group to ensure only one can be selected at a time.
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <RadioGroup value={value} onChange={setValue} name="billingCycle">
          <Radio 
            label="Option 1"
            name="group1"
            value="option1"
          />
          <Radio 
            label="Option 2"
            name="group1"
            value="option2"
          />
        <RadioGroup>
      `} />
    </Canvas.Source>

    ### **How do I make a radio button required?**
    Add the `required` attribute to the radio button input to make it mandatory for form submission
    <Canvas.Source>
      <Source className="my-0!" dark code={`
        <Radio 
          label="Required Option"
          name="requiredGroup"
          value="requiredOption"
          required
        />
      `} />
